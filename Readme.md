# WinWin
## 서비스 목적
프랜차이즈 점주들과 예비 점주들을 도와줄 수 있는 플랫폼 프랜차이즈 점주들과 예비 점주들이 사용할 수 있는 다양한 기능을 제공

## 서비스 타겟
프랜차이즈 점주, 예비 점주, 브랜드 회사들

## 핵심가치
- 예비 점주들이 창업 대한 상담을 받고 정보를 수집할 수 있는 공간
- 점주들이 서로 소통하고, 정보를 교화할 수 있는 공간
- 브랜드 운영자들이 점주 또는 예비 점주에게 효과적으로 홍보할 수 있는 공간


## 서비스 기능
계획한 기능은 다음과 같으나, 이번 프로젝트에서는 일부만 구현
- 프랜차이즈 점주와 예비 점주들이 사용할 수 있는 다양한 기능 제공
    - 점주 커뮤니티
    - 예비 점주가 AI를 통해 프랜차이즈 창업 상담을 받을 수 있는 서비스
    - 프랜차이즈 점주가 본인 브랜드에 평점을 남길 수 있는 기능
    - 프랜차이즈 점주들을 위한 대시보드 기능
    - 프랜차이즈 점주들을 위한 직원 관리 및 교육 기능
- 브랜드 창업자가 효과적으로 홍보할 수 있는 기능 제공

## 구현 완료한 기능
- 메인 뷰 가게 리스트
- 브랜드 상세 페이지
- AI 기반 예비 점주 상담 기능
- 점주 커뮤니티
